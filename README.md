# 🏭 Industrial Wastewater Treatment Optimization Dashboard

End-to-End **Machine Learning** powered web application that helps plant operators and engineers optimize industrial wastewater treatment processes in real-time.

## ✨ Features

- Real-time prediction of **Suspended Solids (SS-S)** in effluent
- Interactive sliders to adjust key process parameters (pH, flow rate, solids, etc.)
- Instant quality classification (**Good Effluent** / **Needs Optimization**)
- Interactive visualizations using Plotly
- Feature importance analysis
- Clean, professional multi-tab interface

## 🛠️ Technologies Used

- **Python**
- **Streamlit** (Web Dashboard)
- **Scikit-learn** — Random Forest Regressor
- **Pandas** & **NumPy**
- **Plotly** (Interactive Charts)
- **Joblib** (Model Saving & Loading)
- UCI Water Treatment Plant Dataset

## 🚀 How to Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/Praveensaitalpuri/Industrial-water-optimizer.git

# 2. Go to project folder
cd Industrial-water-optimizer

# 3. Install dependencies
pip install streamlit pandas plotly scikit-learn joblib

# 4. Run the dashboard
streamlit run app.py
